// Notas del flow actual (2026-05-20)
· El flujo principal ganó 3 pantallas desde el navigator r1: S.ERA_SEL (ahora el estado inicial), S.INTRO (carrusel de storytelling) y S.CLIENT_SEL (mapa de verticales, antes embebido en el menú). Además hay un splash HTML de entrada con JUGAR / DEMO.
· Orden: SPLASH → ERA_SEL → PRELOAD → MENU → INTRO → CLIENT_SEL → MODEL_SEL → TUT → GAME → YEAR_END → GAME_END.
· ?play=creator sigue siendo la entrada canónica de la rama Creator. El modal de plantillas ahora muestra miniaturas de preview del layout en cada tarjeta (era una idea pendiente en r1, ya implementada).
· Capturas de YEAR_END y GAME_END generadas con datos de ejemplo para ilustrar la pantalla.
